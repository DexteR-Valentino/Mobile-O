## 📄 About Mobile-O

Mobile-O provides a way to handle images and text together on your mobile device. It lets you understand images, create new images, and turn images into words or words into images.  

This makes Mobile-O useful for tasks like editing pictures, generating new images from descriptions, or reading text from pictures. It works on devices like smartphones and tablets, giving you powerful tools without needing a desktop computer.

---

## 🖥️ System Requirements

Make sure your Windows PC meets these minimum requirements before running Mobile-O:

- Windows 10 or later (64-bit)
- 4 GB RAM or more
- At least 500 MB free disk space
- Internet connection for initial setup and updates
- A modern graphics card with basic support for machine learning acceleration (optional but helpful)

---

## 🎯 Main Features

- **Image Editing:** Modify pictures easily, including changing colors and removing objects.
- **Image-to-Text:** Extract text from images using optical character recognition.
- **Text-to-Image:** Generate new images from short text descriptions.
- **Unified Interface:** Use all features from the same app without switching tools.
- **Mobile Friendly:** Designed to work smoothly on smaller screens and touch controls.

## ⚙️ Using Mobile-O for the First Time

When you open Mobile-O, you will see a simple interface with buttons to choose what you want to do.

- To **edit an image**, click "Edit Image" and load a photo from your device.
- To **extract text**, select "Image to Text" and choose your picture.
- To **generate a new image**, click "Text to Image" and write a short description.

You can save your work at any time by clicking the save button. The app supports common image formats like JPG and PNG.

## 🔧 Troubleshooting Tips

- **Installer does not run:** Make sure you downloaded the full file. Try right-clicking it and select “Run as administrator.”
- **App won’t start:** Check if your Windows meets the requirements. Restart your computer before trying again.
- **Images fail to load:** Confirm the file is not corrupted. Mobile-O supports JPG, PNG, BMP, and GIF formats.
- **Text extraction errors:** Images must have clear and readable text. Try cleaning or retaking the photo.
